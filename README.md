# Islam & Sciences | Divine Science & The Islamic Golden Age

Explore the harmony between Islam and Science. A resource for Tawheed-based knowledge, the scientific heritage of Islamic civilization, and Divine Laws.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
